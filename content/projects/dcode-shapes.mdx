"Shapes" is a project that allows you to create flowcharts and diagrams with different shapes. It is based on the ReactFlow project.
<a href="https://dcode-store.github.io/shapes/">Try Shapes</a>

## Features

- Create various types of charts
- Export charts as images (SVG, PNG, JPEG)
- Responsive design
- Easy-to-use interface
